StereoQ Player builds and updates its music database using these sources:
  • USB
    Local storage (SATA HDD / SSD). The section is historically called “USB” in Volumio-based systems.
  • WebRadio
    Locally saved web radio playlists.
  • NAS
    Connected network shares (SMB/CIFS, NFS, etc.).
