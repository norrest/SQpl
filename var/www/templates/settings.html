Player Name

Note: the name is shown for reference. You can select and copy it, but editing is disabled.

Services

Enable or disable selected functions.


AirPlay lets you stream audio to the player from Apple devices.

$_system_select[shairport1] $_system_select[shairport0]
Enables shairport-sync. Turn OFF if you never use AirPlay.

Allows remote control of the player using UPnP controller apps (Play To).

$_system_select[upnpmpdcli1] $_system_select[upnpmpdcli0]
Enables upmpdcli (MPD UPnP renderer/control). Turn OFF if you don’t use UPnP apps.

Shows other UPnP/DLNA servers in the network and can mount them into the player as a folder (usually under /mnt/UPNP). This may create extra background traffic on SSDP (UDP 1900) and can be heavy on low-power devices.

$_system_select[djmount1] $_system_select[djmount0]
Enables djmount. If you don’t use UPnP libraries, keep this OFF.

Shares your local music folder to the network as a DLNA server. Other devices (TV, phone apps, streamers) will see the player as a media library. Can increase disk activity and CPU usage during scanning.

$_system_select[minidlna1] $_system_select[minidlna0]
Enables minidlna.
Misc

Useful system options.

$_system_select[startupsound1] $_system_select[startupsound0]
Plays a short sound during boot. Turn OFF for silent startup.
Sound Quality

Kernel profiles adjust low-level system parameters. ECO mode may reduce heat and power use.

Default is balanced. ECO reduces CPU load and heat on weak hardware.
Compatibility

Fixes for specific USB receivers and DACs.

$_system_select[cmediafix1] $_system_select[cmediafix0]
For CM6631-based devices that produce clicks/noise between tracks with different sample rates. Use only if you have the issue.
